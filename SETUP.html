üêõ Troubleshooting

Movies not showing?

  • Check folder path in appsettings.json
  • Ensure videos have supported extensions
  • Click "Scan Movies" again

Video won't play?

  • MKV files may need conversion to MP4
  • Ensure H.264 codec for MP4 files

Subtitles not working?

  • File must be named same as video (movie.mp4 ‚Üí movie.srt)
  • Only .srt format supported
  • Use UTF-8 encoding
